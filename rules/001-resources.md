# Правило 001. О ресурсах

> Уровень: Правила (51%+ голосовавших, 3 дня cooling, x1 залог)
> Статус: драфт

---

## Раздел 1. Классификация ресурсов

1.1. Ресурсы делятся на два типа по возобновляемости:
     - **Конечные** — мировой запас уменьшается при добыче, не восстанавливается.
     - **Возобновляемые** — восстанавливаются каждый тик по формуле.

1.2. Ресурсы делятся на три категории по происхождению:
     - **Сырьё** — добывается из мира напрямую.
     - **Материалы** — производятся из сырья.
     - **Продукция** — производится из материалов и/или сырья.

## Раздел 3. Добыча сырья

3.1. Каждый тик гражданин может подать одну заявку на добычу.
3.2. Базовая добыча без инструментов:

| Ресурс | Количество за тик |
|--------|-------------------|
| wood | 5 |
| stone | 3 |
| grain | 8 |
| ore | 2 |
| clay | 6 |

3.3. Инструмент (tool) увеличивает добычу в 2 раза.
     При использовании tool изнашивается: -1 прочность за тик.
3.4. Новый tool имеет прочность 10 (хватает на 10 тиков).
3.5. Если мировой запас конечного ресурса = 0, добыча невозможна.
3.6. Если запрошено больше чем осталось — добывается остаток.

## Раздел 4. Восстановление возобновляемых ресурсов

4.1. Каждый тик мировой запас возобновляемого ресурса увеличивается
     на значение из таблицы (раздел 2), но не выше начального запаса.
4.2. Формула: `новый_запас = min(запас + восстановление, начальный_запас)`

## Раздел 5. Хранение

5.1. Ограничения на хранение ресурсов у гражданина нет.
5.2. Ресурсы не портятся и не исчезают (кроме износа инструментов).
