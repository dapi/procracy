# Закон 003. О банкротстве

> Уровень: Закон (60%+ голосовавших, 5 дней cooling, x2 залог)
> Статус: драфт

---

## Раздел 1. Определение банкротства

1.1. Гражданин считается банкротом, когда баланс его merits
     и суммарная стоимость ресурсов < 0 (с учётом обязательств).
1.2. Сервер автоматически определяет статус банкротства каждый тик.

## Раздел 2. Последствия банкротства

2.1. Банкрот сохраняет:
     - Гражданство и право голоса.
     - Право подавать PR уровня "Код" (без залога).
     - Право на получение ресурсов от других игроков.
     - Право на производство и торговлю.

2.2. Банкрот временно теряет:
     - Право подавать PR уровней "Правила", "Законы", "Конституция"
       (требуют залога, который банкрот не может внести).

## Раздел 3. Выход из банкротства

3.1. Статус банкротства снимается автоматически, когда баланс
     становится >= 0.
3.2. Других условий для выхода из банкротства нет.

## Раздел 4. Защита кредиторов

4.1. Обязательства банкрота не списываются. Долг сохраняется
     до погашения.
4.2. Порядок погашения: пропорционально всем кредиторам.
4.3. Принудительное взыскание: кредитор может подать PR уровня "Правила"
     на списание долга. При принятии — долг списывается, убыток
     остаётся за кредитором.
